[![gpuvoxels](example/screenshot.png)](https://gpuvoxels.gatunes.com)

### Dev environment

```bash
# clone repo:
git clone https://github.com/danielesteban/gpuvoxels.git
cd gpuvoxels
# install dependencies:
npm install
# start environment:
npm start
# open http://localhost:8080/ in your browser
```
